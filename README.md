## ğŸš€ Getting Started

### Prerequisites

- Node.js
- MongoDB Atlas Account
- Razorpay Account

---

### Installation

1. Clone the repository:
  
   git clone https://github.com/VEERA5603/WorkHub.git
   cd WorkHub
  

2. Install dependencies:

   npm install
 

3. Configure variables:
   
   MONGODB_URI=<your-mongodb-atlas-uri>
   

4. Start the server:
  
   npm start
  

5. Open the app in your browser:

   http://localhost:4000
